Getting Started

Installation

1
Download the installer Download LED Pixel Mapper Setup 1.5.x.exe from your Gumroad purchase link or the website.
2
Run the installer Double-click the EXE. It installs silently — no wizard, no UAC prompt. It installs per-user so no admin rights are needed.
3
Launch the app Find LED Pixel Mapper in your Start Menu or desktop shortcut and launch it.
ℹ️
Updating Just run the new installer over the existing install — no uninstall needed. Your projects, settings, and license key are preserved automatically.

First Launch

On first launch you will see the Project Setup screen. This is where you define your canvas size, tile profiles, and processors before starting to build. The app starts in Free tier mode until you activate a Pro license.

System Requirements

  • OS: Windows 10 or Windows 11 (64-bit)
  • RAM: 4 GB minimum, 8 GB recommended
  • Display: 1280×720 minimum, 1920×1080 recommended
  • Internet: Required for license activation only — the app works fully offline after that
2

The Interface

The builder is divided into five main zones:

🔧 Top Toolbar

Undo, Redo, mode switcher (Select / Port Routing / Circuit Routing), snap controls, zoom controls, and the label visibility toggle.

📦 Left Sidebar

Your tile profile library. Click any tile to add it to the canvas. Search, filter by brand, and add tiles from the full hardware library.

🖼️ Canvas

The pixel-accurate work area. Drag tiles to position them. Pan with right-click drag. Zoom with Ctrl+scroll or the toolbar buttons.

⚙️ Right Sidebar

Properties panel for the selected tile — name, position, rotation, lock toggle, and individual label controls.

🔌 Bottom Panel

Switches between Port Panel (data routing) and Circuit Panel (power mapping). Also shows bulk edit options when multiple tiles are selected.

📁 File Menu

Top-left button. New project, Open, Save, Save As, language switcher, and license activation.

Wall Tabs

Along the top of the canvas you'll find the Wall Tab Bar. Each tab is a separate wall in your project. Click a tab to switch walls. Click + Wall to add a new wall. Double-click a tab to rename it.

💡
Inactive walls are always visible Tiles on other walls appear as semi-transparent outlines on the canvas so you can see how everything relates spatially.
3

Creating a Project

Project Name

Enter a descriptive name for your project — this appears on exports and saved files.

Canvas Resolution

The canvas represents the total output resolution of your video processor. Choose from common presets or select Custom to enter any width and height in pixels. You can also type math expressions — e.g. 3840/2 evaluates to 1920.

  • 1280 × 720 — HD
  • 1920 × 1080 — Full HD (most common)
  • 2560 × 1440 — QHD
  • 3840 × 2160 — 4K UHD
  • 7680 × 4320 — 8K UHD
  • Custom — any resolution

Bit Depth Pro

Choose 8-bit or 10-bit. 10-bit mode halves the effective pixel capacity of each processor port. This affects the port capacity displayed in the Port Panel and exports.

Tile Profiles

Tile profiles define the LED panels you're using. Each profile has a name, pixel width, pixel height, and optional power draw.

From the Hardware Library

1
Click "From Library" Opens the searchable hardware library with hundreds of real tiles from ROE Visual, Absen, Brompton, Novastar, and more.
2
Search or browse by brand Type a model name to filter, or collapse/expand brand groups. Click + Add next to any tile to add it to your project.

Custom Profile

Click + Add Custom and fill in the name and pixel dimensions manually. Use this for unlisted or custom LED panels.

ℹ️
Free tier limitation Free users can only add tiles from ROE Visual and Absen. All other brands require Pro.

Processors

Add the video processors you'll be using. Click + Add Processor to open the picker:

  • Select the model from the dropdown
  • For XD-expander models (e.g. Brompton SX40), select how many XD boxes are connected
  • Click + Add to add it to your project
  • Each processor is given a default name — double-click it in the list to rename

Returning to Setup (Editing a Project)

Click the LPM logo button in the top-left corner of the builder at any time to return to the Setup screen and adjust canvas size, bit depth, or add more tile profiles and processors. Nothing is lost — all existing walls and tiles are preserved. New profiles and processors are appended.

Working with Tiles

Adding Tiles to the Canvas

In the Left Sidebar, your tile profiles are listed. Click any tile card to instantly place it on the canvas. It drops near the center — drag it into position from there.

You can also add tiles directly from the full hardware library inside the builder by clicking Browse Library at the top of the left sidebar.

Selecting Tiles

  • Single click — select one tile
  • Ctrl + click — add/remove a tile from the selection
  • Click and drag on empty canvas — drag-select rectangle
    • Drag left to right — selects only tiles fully inside the box
    • Drag right to left — selects any tile that the box touches
  • Click empty canvas area — deselect all

Moving Tiles

  • Drag — drag any tile to move it
  • Arrow keys — nudge selected tiles 1px at a time
  • Shift + Arrow keys — nudge 10px at a time
  • X / Y position inputs — type exact coordinates in the HUD that appears when tiles are selected (bottom of canvas)

Snap Controls

Use the snap buttons in the Top Toolbar to control snapping behavior:

  • Snap to Grid — tiles snap to a pixel grid. You can set the grid size in pixels.
  • Snap to Tile — tiles snap to the edges of other tiles for seamless butting.
💡
Both snaps work together Enable both for the most precise placement — tile edges will snap to each other and also align to the grid.

Rotation Pro

Select one or more tiles and:

  • Press R — rotate 90° clockwise
  • Press Shift + R — rotate 90° counter-clockwise
  • Use the rotate buttons in the Right Sidebar (single tile) or Bulk Panel (multiple tiles)

Locking Tiles

Lock a tile to prevent it from being accidentally moved. Select a tile and click the 🔒 Lock button in the Right Sidebar, or use the lock button in the Bulk Panel for multiple tiles. Locked tiles show a small gold lock badge and cannot be dragged.

Renaming Tiles

Select a tile and click on its name in the Right Sidebar to rename it. This name appears as the label on the canvas and in exports.

Deleting Tiles

  • Select tiles and press Delete or Backspace
  • Or click the ✕ Delete button in the Right Sidebar

Copying and Pasting

  • Ctrl + C — copy selected tiles
  • Ctrl + V — paste (tiles appear slightly offset from original)

Undo / Redo Pro

  • Ctrl + Z — undo last action
  • Ctrl + X — redo
  • Or use the undo/redo buttons in the Top Toolbar

Label Visibility

Click the Labels button in the Top Toolbar to toggle which labels are shown on tiles. You can show or hide any combination of:

  • Tile name
  • Port label and chain order (e.g. A1, A2…)
  • Circuit label
  • Tile number
  • Port number and chain order (raw)
  • Power position and wattage
  • Tile resolution (pixel width × height)
5

Managing Walls

Adding a Wall

Click + Wall in the Wall Tab Bar. Free users are limited to 2 walls. Pro users can add unlimited walls.

Switching Walls

Click any wall tab to make it the active wall. The canvas shows that wall's tiles as fully opaque. All other walls appear as faded outlines.

Renaming a Wall

Double-click a wall tab to rename it inline.

Deleting a Wall

Hover over a wall tab and click the ✕ button that appears. You cannot delete the last remaining wall.

⚠️
Deleting a wall is permanent All tiles, port assignments, and circuit assignments on that wall will be lost. Use undo immediately if you delete by mistake (Pro only).

Multi-Wall Workflow Tips

  • Name your walls clearly — e.g. "Main Stage", "IMAG Left", "Backdrop"
  • Each wall shares the same processor pool — you can route ports from any processor to any wall
  • Canvas size is the same for all walls — it represents the output resolution of your processor
  • Use the inactive wall outlines to check that tiles on different walls don't overlap in ways that would conflict on the processor output

Data Port Routing

The Port Panel lets you assign tiles to processor output ports. Each port gets a letter name (A, B, C…), a color, and a pixel capacity based on the processor model.

Opening the Port Panel

Click the Ports tab in the bottom panel. You'll see all processors and their ports listed.

Adding a Processor Port

Processors are added during project setup. In the Port Panel you can see all existing ports. To add more processors, return to Setup via the logo button.

Assigning Tiles to a Port

1
Click a port row Click any port in the Port Panel to enter Port Routing Mode. A colored banner appears at the top of the canvas.
2
Click tiles on the canvas Each tile you click is added to that port in chain order. The tile is highlighted with the port color. Chain numbers are assigned automatically.
3
Use arrow keys to navigate After selecting a tile, use Arrow keys to jump to the nearest tile in that direction and assign it next in the chain.
4
Exit routing mode Click the same port again, click a different port, or press Escape to exit routing mode.

Port Capacity

Each port shows its pixel capacity and current usage. The capacity is automatically halved in 10-bit mode. The panel shows a warning if you exceed the port's capacity.

Data Flow Overlay

When a port is active, arrows are drawn on the canvas showing the data chain order between tiles. You can toggle arrows, path lines, and chain number markers using the display controls in the Port Panel.

Clearing a Port Assignment

In the Port Panel, click the ✕ next to any tile in a port's list to remove it from that port. Or clear all assignments for a port using the clear button.

10-bit Mode

Enable 10-bit in project settings. This halves the pixel capacity shown on each port. Use this when running a 10-bit signal chain to accurately calculate how many tiles each port can drive.

Power Circuit Mapping

The Circuit Panel lets you group tiles into power circuits and automatically calculates the total load per circuit.

Opening the Circuit Panel

Click the Circuits tab in the bottom panel.

Creating a Circuit

1
Click "+ Add Circuit" A new circuit is created with a name and an auto-assigned color.
2
Click the circuit row to enter routing mode A colored banner appears on the canvas just like port routing.
3
Click tiles to assign them Each tile you click joins the circuit. Tiles show numbered badges and are connected by colored arrows.

Load Calculation

The Circuit Panel automatically sums the power draw of all tiles assigned to each circuit. Power values come from the tile profiles you set up (in watts). The total is shown per circuit so you can balance loads across power supplies.

💡
Set power values in tile profiles For accurate load calculations, make sure each tile profile has a watt value set. You can add or update this in the Project Setup screen.

Renaming and Deleting Circuits

Click the circuit name in the panel to rename it inline. Click the ✕ button to delete the circuit (tiles are unassigned, not deleted).

Exporting

Click the Export button in the Top Toolbar to open the Export Dialog.

Export Types

PNG — Pixel Map Free

Exports the canvas as a PNG image showing tile positions and pixel boundaries. Used to feed into your video processor's mapping software. Available for the full canvas or individual walls.

PNG — Data Flow Free

Exports the canvas with data port routing arrows overlaid — showing which port connects to which tiles and in what chain order. Great for cable labeling docs.

PNG — Power Flow Free

Exports the canvas with power circuit routes overlaid — showing which tiles belong to which circuit. Essential for power distribution planning.

CSV — Production Report Pro

Exports a comma-separated spreadsheet with a full tile-by-tile breakdown: tile name, position, profile, assigned port, chain order, circuit, and power draw. Open in Excel or Google Sheets.

PDF — Production Report Pro

Exports a multi-page PDF containing all pixel maps, data flow and power flow diagrams, and a summary table. Formatted for printing and handing to your crew.

Export Scope

Choose to export the full canvas (all walls combined) or a single wall at a time.

Logo on Exports Pro

Pro users can upload a company logo that will be embedded in the corner of all PNG and PDF exports. Free exports include a watermark instead.

ℹ️
Free tier watermark All exports in the Free tier include a centered semi-transparent watermark and diagonal "FREE VERSION" text. Upgrade to Pro to remove it.

Saving & Loading

All file operations are in the File Menu — the top-left button labeled "File".

Save

File → Save (or Ctrl+S coming in a future update) saves the project to its current file. The first time you save a new project, it will ask you where to save it. After that, Save writes to the same file silently.

Save As

File → Save As always opens the native Windows save dialog so you can choose a new filename and location. Use this to create a copy of a project under a different name.

Open Project

File → Open Project opens a file picker. Select any .lpmap.json file to load it. This replaces the current project — make sure you've saved first.

New Project

File → New Project clears the current project and returns to the Setup screen. Save your work first.

Project File Format

Projects are saved as .lpmap.json files — plain JSON text. They can be backed up, version-controlled, or shared with other LED Pixel Mapper users.

💡
Back up your project files Save project files to cloud storage (Dropbox, Google Drive, OneDrive) so you never lose work. The file is small — usually under 500 KB even for large builds.

License & Activation

Activating Pro

1
Purchase from Gumroad Buy LED Pixel Mapper Pro at eventpeople.gumroad.com/l/LEDPixelMapper. Your license key is shown immediately after purchase and also emailed to you.
2
Open the activation dialog In the app, click the File menu → Activate License…
3
Enter your license key Paste the key in the format XXXXXXXX-XXXXXXXX-XXXXXXXX-XXXXXXXX and click Activate. An internet connection is required.
4
Done — Pro features are unlocked The app will show "License: Pro ✓" in the File menu. All Pro features are now available.

Machine Limit

Each license key can be activated on up to 2 machines. If you need to activate on a new machine after reaching the limit, contact support to reset your key.

Offline Use

After activation, the app works fully offline. It re-verifies your license in the background every 30 days. If it can't reach the server (no internet), it enters Grace Mode — all Pro features remain fully functional.

License Status Indicator

  • License: Pro ✓ — Active and verified
  • License Overdue ⚠️ — Grace mode — couldn't re-verify but still fully functional
  • No indicator — Free tier

Support

For license issues, email kris.siwak@eventpeople.ca with your purchase email and license key.
